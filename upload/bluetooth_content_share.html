=== STACK TECHNO OBLIGATOIRE 2026 ===
1. Front: Angular 17 Standalone + Angular Material + Tailwind CSS + NgRx State + PWA Workbox + Dexie.js IndexedDB
2. Back Core: Spring Boot 3.3 + Spring Security JWT + Spring WebSocket STOMP + Spring Cloud Gateway + Spring AI + Spring Actuator
3. Back Modules: Laravel 11 API pour Paiement/Bibliothèque/Réservation rapide
4. DB: MySQL 8 Master + 2 Slaves réplication + Redis Cluster + MinIO 4 nœuds distribué S3-like
5. Temps réel: Socket.IO + Redis Pub/Sub pour chat/notifs
6. IA: Ollama Llama 3.1 local + Whisper pour sous-titres + RAG sur PDF cours pour CampusBot
7. Map: OpenStreetMap + Leaflet + Geolocation API
8. DevOps: Docker + Docker Compose + Nginx Load Balancer + Healthcheck + Auto-scaling
9. Tests: k6 charge 10k users + Axe-core accessibilité + Lighthouse >95
